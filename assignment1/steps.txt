git clone link : repo klonlandi
git branch feature: feature adinda yeni bir branch olusturuldu
git checkout feature: feature branche gecis yapildi
git branch: hangi branchte oldugumuz kontrol edildi
nano feature.txt: feature.txt olusturuldu ve icine content eklendi
git add . : tum degisiklikler eklendi
git commit -m "add feature.txt": tum degisiklikler commitlendi
git push origin feature: feature branche commit pushlandi
create pr and merge from github: pr olusturuldu ve mergelendi
git checkout main: main branch e gecildi
git branch -d feature : feature branch silindi
git branch: branchler kontrol edildi
